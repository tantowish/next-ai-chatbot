**Clone**

```
git clone https://github.com/tantowish/zenspire-web.git
cd zenspire-web
```

**Install local dependencies**

```
npm install
```

**Running Service**

```
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
